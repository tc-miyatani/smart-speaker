Raspberry Pi（ラズベリーパイ）
	ARMプロセッサを搭載したシングルボードコンピュータ。

・オームの法則
	E = I R
	E(V):電圧。水路の高低差
	I(A):電流。水路に流れる水の量
	R(Ω):抵抗。水路の大きさ
・キルヒホッフの法則
	第1法則
		電気回路の任意の分岐点について、そこに流れ込む電流の和は、そこから流れ出る電流の和に等しい。
		並列回路においてI = I1 + I2
	第2法則
		電気回路の任意の一回りの閉じた経路について、電位差の和は 0 である。
		直列回路においてV = V1 + V2, 並列回路に置いてV = V1 = V2

・GPIO
	汎用的なピン。プログラム制御により、ON/OFF・入力/出力が設定できる。
・GND(グランド)
	接地。アースとも呼ばれる。マイナス極側を繋ぐ。

・コンデンサ	
	英語ではCapacitor(キャパシタ)といい、記号はC。容量を示す単位はF（ファラッド）。
	抵抗と異なり、電気を貯める性質があり、容量（静電容量）という言葉を使う。
	C=Q/V
	C(F):静電容量	
	Q(C)：電荷
	V(V):電圧
	直列回路において1/C = 1/C1 + 1/C2, 並列回路においてC = C1 + C2
・トランジスタ
	電流の制御（増幅）を行う。
	電流伝送率α=Ic/Ie、直流電流増幅率β(hfe)=Ic/Ib


Vcc:コレクタ（C）側に供給
Vee:エミッタ（E)側に供給

P(W) = V(V) × I(A)

R = (v - VF) ÷ IF 

●トランジスタ
・スイッチング
	VB < 0.6VでCE間(Ic)に電流が流れない
	VB > 0.6VでCE間(Ic)に最大Vcc/RLの電流が流れる(オームの法則)
・電流の増幅
	Ic = Ib * hfe

＜問題点＞
・CE間に0.2V程度の電圧が残る
・VBはo.7V程度、数Vの電圧をかけるとエミッタ電流が以上に大きくなる
	→	入力電源が5Vの場合、ベースやエミッタに抵抗を入れる。
	　　	トランジスタをスイッチとして使う場合、エミッタに抵抗を入れず、ベースに入れる。
		IB = Ic / hfe = Vcc/RL * 1/hfe
		VB = IB * RB + 0.7
		ex) IB = 0.5A / 100 = 5mA
			RB = (5V - 0.7V) / 5mA = 860Ω

・エミッタ電流が大きいとIc = Ib * hfeの割合で大きいコレクタ電流を流そうとするが、
　Vcc/RL以上のコレクタ電流は流れないので、
　超えた分がベース電流となって流れ、トランジスタを破損する

・hfeのバラツキ
	hfeにはバラツキがある　→　hfeの最小値を使ってベース電流を求め、その３倍程度のベース電流を流す（オーバードライブ）。
・遮断電流
	スイッチがOFFでも、実際には数μA程度の遮断電流(ICBO)が流れ、コレクタ端子にはICBO * hfeの電流が流れ、スイッチのOFF状態を崩してしまう。
	→	BE間に1kΩ程度の抵抗RBEを接続し、遮断電流が出来るだけベース端子に流れ込まないようにする。
		RBEがあるとRBEにも電流が流れる為、トランジスタをONにするためには更に大きな電流Isを流す必要が生じる。
		Is = IB + IRB = IB + 0.7/1 = IB + 0.7mA
・スイッチング速度
	ON/OFF切り替え後、コレクタ電流はほんの少し遅れて流れる。
	→	コンデンサを接続することによってスイッチング速度を早められる。
	
	

●PWM

PWM(Pulse Width Modulation、パルス幅制御)とは、
一定電圧から、オンとオフの一定周期を作り、オンの時間幅を変化させる事です。
一般的なマイコンは、出力電圧を変化させる事が出来ないので電圧を時間で刻んで電圧が下がったと見せかけます。

一定の繰り返し時間：PWM周期(パルス期間、パルス周期)
ONタイム：デューティー(パルス幅)
Duty比％(D) = パルス幅(τ) / PWM周期(T) * 100
	ex) 5V電圧でデューティー比0%→0V, 50%→2.5V, 100%→5V
PWM周波数 = 1 / PWM周期
	ex) 1 / 1.25ms = 800Hz = 0.8kHz
	    1 / T = 36kHz → T = 26μs
